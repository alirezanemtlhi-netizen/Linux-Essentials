* **Software Property:** Software is a financial asset governed by legal rules.
* **Copyright:** Legal ownership protecting source code, algorithms, names, and logos.
* **Software License:** A legally binding contract defining how to use, modify, and share code.
* **FSF (Free Software Foundation):** Promotes user freedom (GNU Project). Free means liberty, not $0 cost.
* **OSI (Open Source Initiative):** Promotes open-source development for commercial business.
* **CC (Creative Commons):** Licensing framework for creative media, text, and images.
* **Business Models:** Open-source companies make money via paid Support and Dual-Licensing.
* **GPL License:** The Linux Kernel license. Modified code must remain open-source (Copyleft).
* **Apache License:** A permissive license. Allows using open code in closed proprietary software.
* **GitHub:** The ultimate global directory and reference source for open-source software.
* **Open Source Tools:** WordPress (Websites), Zabbix (Monitoring), pfSense (Firewall), Proxmox (Virtualization).
* **Proprietary Example:** Google Analytics (Closed-source software owned by Google).

# Linux Essentials — Day 3
## Chapter 3: Linux Philosophy and Concepts

* **Exam Target:** 1 Question from this chapter.
* **Birth of Linux:** Created in 1991 by Linus Torvalds as a fun project.
* **Before Linux:** People used MS-DOS (Single-user, single-task interface).
* **UNIX Design:** Linux is based on UNIX to support multi-user and multitasking.
* **Evolution:** Linux changed from a simple emulator into a full OS kernel.
* **Early Distro:** Red Hat was one of the first commercial distributions.
* **Monolithic Kernel:** Linux architecture. It runs all OS services inside one big process for high performance.
* **Microkernel:** Newer architecture. It keeps the core kernel small and uses separate modules.
* **Modern Linux:** Supports over 32 CPU architectures, power management, and advanced network drivers.
* **Ecosystem:** Has diverse GUIs, 10,000+ tools, and many distributions.
* **Open Source vs Closed Source:** Difference in how you get it, what you can do with it, and how it updates.
* **Open Source Pros:** Better code (global review), high flexibility (customizable), and lower costs.
* **Linux Roles:** Used in Embedded Systems (smart devices), PCs (laptops), and Server Systems (internet backbone).
